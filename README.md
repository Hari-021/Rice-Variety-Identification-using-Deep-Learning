# 🌾 Rice Variety Identification Using Deep Learning

> A Deep Learning–based web application that classifies rice varieties from grain images using Convolutional Neural Networks (CNNs) and Transfer Learning.

## 📖 Overview

This project leverages **Deep Learning** and **Transfer Learning** techniques to automatically identify different rice varieties from images. It aims to assist researchers, farmers, and quality inspectors by providing a fast and accurate classification system.

---

## ✨ Features

* 🌾 Automatic rice variety classification
* 📷 Image upload and prediction
* 🤖 CNN and Transfer Learning–based model
* 🌐 Simple and responsive web interface
* ⚡ Fast inference and accurate results
* 📊 Suitable for educational and research purposes

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Hari-021/Rice-Variety-Identification-using-Deep-Learning.git
```

### Navigate to the project

```bash
cd Rice-Variety-Identification-using-Deep-Learning
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```

## 📸 How It Works

1. Upload a rice grain image.
2. The image is preprocessed.
3. The trained deep learning model analyzes the image.
4. The predicted rice variety is displayed to the user.
